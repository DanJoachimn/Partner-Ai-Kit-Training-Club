## Benchmark files

Format:

```markdown
---
type: benchmark
generated_by: claude-code
name: "5k Row Test"
purpose: "aerobic baseline + Z2 pace anchor"
frequency: "every 3 weeks during base/build"
---

## The Test
Row 5,000m for time. Standard rower. Damper at coach's call.

## Notes for athletes
- This is a sub-max effort. We want a sustainable Z3 pace, not a max.
- Splits: 500m splits should be within 5 seconds of each other.
- Recovery: light spin / mobility for the rest of the session.

## Notes for coaches
- Record split times to each athlete's `Members/[name].md` under "Recent benchmarks."
- Trend across 3+ tests is more informative than any single result.
```
